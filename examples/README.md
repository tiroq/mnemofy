# mnemofy Output Examples

This directory contains example output files generated by the mnemofy CLI.

## Files

- **sample.transcript.txt** - Plain text transcript with timestamps
- **sample.transcript.srt** - SubRip subtitle format for video players
- **sample.transcript.json** - Structured JSON with metadata
- **sample.notes.md** - Structured meeting notes with 7 sections

## Usage

These examples show what mnemofy generates from a sample meeting recording.

To generate similar outputs from your own audio/video:

```bash
mnemofy transcribe your_file.mp4
```

You'll get all 4 output formats automatically.

## Format Details

### TXT Format
- **Purpose**: Reading, searching, printing
- **Format**: `[HH:MM:SSâ€“HH:MM:SS] text`
- **Best for**: Quick review, text processing

### SRT Format  
- **Purpose**: Video subtitles
- **Format**: SubRip standard (sequence #, timing, text, blank line)
- **Best for**: VLC, video editors, subtitle tools

### JSON Format
- **Purpose**: Programmatic access
- **Structure**: `{schema_version, metadata, segments[]}`
- **Best for**: Data analysis, custom processing

### Notes Format
- **Purpose**: Structured meeting summary
- **Sections**: Metadata, Topics, Decisions, Actions, Mentions, Risks, File Links
- **Best for**: Team sharing, quick review
